# Tech Stack: Agili's Hacker Podcast

## Frontend
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS, shadcn/ui
- **State Management:** Zustand
- **I18n:** Custom i18n implementation (JSON-based)

## Backend & Infrastructure
- **Runtime:** Cloudflare Workers / Pages
- **Deployment:** OpenNext, Wrangler
- **Storage:** Cloudflare R2 (Audio/Assets), Cloudflare KV (Metadata/Settings)

## Languages & Tooling
- **Language:** TypeScript
- **Package Manager:** pnpm
- **AI/ML:** TTS and LLM integration for summarization and audio generation
